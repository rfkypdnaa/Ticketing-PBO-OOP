# Ticketing-PBO-OOP
Ticketing-PBO merupakan laporan/tutorial aplikasi Ticketing yang disusun sebagai bagian dari tugas mata kuliah Basis Pemrograman Object. Dokumen ini menjelaskan alur penggunaan dan tampilan antarmuka (UI) aplikasi ticketing secara bertahap melalui gambar dan penjelasan singkat.
Halaman Login 
Menampilkan halaman login yang mewajibkan pengguna memasukkan username dan password sebagai proses autentikasi sebelum dapat mengakses aplikasi.

Dashboard 
Setelah login berhasil, pengguna diarahkan ke dashboard yang terdiri dari:

Header: Menampilkan informasi utama aplikasi

Sidebar: Navigasi menu

Main Body: Area utama untuk menampilkan konten

Master Harga Ticket 
Berisi daftar master harga tiket yang digunakan sebagai referensi dalam transaksi pembelian.

Transaksi Pembelian Ticket
Menampilkan daftar transaksi pembelian tiket yang telah dilakukan oleh pengguna.

Menu POS Ticketing 
Digunakan untuk melakukan proses penjualan tiket secara langsung (Point of Sale).

Output Pembelian Ticket 
Menampilkan hasil/output dari transaksi pembelian tiket yang telah diproses.

Transaksi Check In 
Menampilkan daftar dan menu transaksi check-in ticket, yaitu proses validasi tiket saat digunakan.

Transaksi Check Out 
Menampilkan daftar dan menu check-out ticket, sebagai penanda bahwa tiket telah selesai digunakan.

Report Transaksi 
Menyediakan laporan seluruh transaksi yang dapat difilter sesuai kebutuhan pengguna.
